Admin setup guide  ▸
1
Host this file Deploy puzzel-agent-widget.html to any HTTPS URL (e.g. your CDN, SharePoint, or a simple web server).
2
Create the widget in Admin Portal Go to Administration → Widgets → Widget Administration and create a new Custom Widget. Set the url property to your hosted URL. Optionally set handshake to true to ensure the widget signals readiness before events are sent.
3
Add to a Tab In Widgets → Tab Configuration, add the widget to a new or existing Request Tab. Enable Default opened in left tab-set so it auto-opens when a call arrives.
4
Pass IVR variables In your IVR flow, use kml_-prefixed variables (e.g. kml_accountRef, kml_enquiryType) to pass data from the caller to the agent screen. This widget reads them from the requestAccepted event payload.
5
Widget origin The Widget API channel is opened from https://agent.puzzel.com. This is set in the AGENT_APP_ORIGIN constant at the top of the script — no changes needed for production.
